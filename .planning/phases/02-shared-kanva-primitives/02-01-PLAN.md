    Parameters (via `{% render %}` named params):

    - `alignment` (string, default 'left') — 'left', 'center', 'right'

    HTML structure:
    ```
    <div class="kanva-heading" style="text-align: {{ alignment | default: 'left' }};">
      {% if label != blank %}<p class="kanva-heading__label">{{ label }}</p>{% endif %}
      {% if heading != blank %}<{{ heading_tag }} class="kanva-heading__title">{{ heading }}</{{ heading_tag }}>{% endif %}
      {% if subtext != blank %}<p class="kanva-heading__subtext">{{ subtext }}</p>{% endif %}
    </div>
    ```

    Include a `{%- doc -%}` block at the top documenting all parameters.

    CSS (in a `<style>` tag at the bottom of the snippet):
    - `.kanva-heading__label`: font-size 12px, font-weight 500, letter-spacing 0.08em, text-transform uppercase, color var(--kanva-text-muted), margin-bottom 16px

    Parameters:
    - `image` (image object, optional) — Card image
    - `image_alt` (string, optional) — Alt text for the image
    - `url` (string, optional) — Card link URL
    - `badge_text` (string, optional) — Badge text rendered via kanva-badge
    - `title` (string, optional) — Card title text
    - `subtitle` (string, optional) — Card subtitle/description

    HTML structure:
    ```
    <div class="kanva-card">
      <div class="kanva-card__media">
        {% if url != blank %}<a href="{{ url }}" class="kanva-card__link">{% endif %}
          {% if image != blank %}
            {{ image | image_url: width: 600 | image_tag: alt: image_alt, loading: 'lazy', class: 'kanva-card__img' }}
          {% else %}
            {{ 'image' | placeholder_svg_tag: 'kanva-card__img kanva-card__placeholder' }}
          {% endif %}
          {% if badge_text != blank %}
            <div class="kanva-card__badge">{% render 'kanva-badge', text: badge_text %}</div>

      {% if title != blank or subtitle != blank %}
        <div class="kanva-card__body">
          {% if title != blank %}<h3 class="kanva-card__title">{{ title }}</h3>{% endif %}

        </div>
      {% endif %}
    </div>
    ```
